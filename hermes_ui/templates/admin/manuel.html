Principes généraux

1 - Introduction

1.1 - Objectif du projet

Ce programme permet à une personne ou à une organisation d'automatiser le traitement de message electronique ayant une structure normale et répétitive.

Créez de la valeur en diminuant drastiquement la contrainte de votre activité. Dans le cadre de la gestion de gros volume de message électronique.

1.2 - Les grands principes

Notre solution s'articule entre ces quatres élements principaux :

  • Règles de détection
  • Détecteur
  • Automate
  • Action

1.3 - Définitions

Une règle de détection : Critère à tester dans la source dont la validation est requise ou non.
Un Détecteur : Un détecteur rassemble une collection de règles de détection et permet à fortiori de déterminer la nature d'une source.
